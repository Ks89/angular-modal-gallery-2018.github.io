UpperButtons

UpperButtons component contains buttons receiving a ButtonsConfig as input


ButtonsConfig

Interface to customize modal upper buttons.


visible
Type: boolean

If false, all buttons are hidden, otherwise not.
strategy
Type: ButtonsStrategy

Enum to decide which buttons are visible, based on some patterns (default, simple, advanced...)
buttons
Type: ButtonConfig[]optional

If you choose ButtonsStrategy.CUSTOM you can add your buttons to this array

ButtonEvent

Interface that represent a button event.


button
Type: ButtonConfig

The button configuration / representation
image
Type: Image | null

The current Image object
action
Type: Action

It's the enum Action

ButtonConfig

Interface that represent a button.


className
Type: stringoptional

To specify the css class name with the icon (for instance, with Font Awesome, you can use 'fa fa-close')
size
Type: Sizeoptional

To choose the size of the button (if you are using fonts as icons please check 'fontSize'). width: 30px and height auto by default
fontSize
Type: stringoptional

To specify the size of the font ('font-size' css property) if you are using a font as icon (for instance, with Font Awesome)
type
Type: ButtonType

Enum to specify the type of the button. If you want to create a custom button, use ButtonType.CUSTOM
title
Type: stringoptional

Optional title text. It sets 'title' html's attribute, useful for accessibility purposes. To ignore this feature, don't add this property
arialLabel
Type: stringoptional

Optional aria label text. It sets 'aria-label' html's attribute, useful for accessibility purposes. To ignore this feature, don't add this property
extUrlInNewTab
Type: booleanoptional

To open in a new tab (instead of the current one) the url specified in extUrl property

ButtonsStrategy

Enum to decide which buttons are visible or not. It can be one of these enum values:


  • DEFAULT: only close button
  • SIMPLE: close and download button
  • ADVANCED: close, download and exturl (to open a link in the current tab)
  • FULL: close, download, exturl (to open a link in the current tab), delete and full-screen
  • CUSTOM: to add an array of ButtonsConfig

ButtonType

Enum that represent the button's type. It can be one of these enum values:


  • DELETE: delete button
  • EXTURL: extUrl button (to open a link in the current tab)
  • DOWNLOAD: download button
  • CLOSE: close button
  • CUSTOM: custom button created by you from scratch (behaviour, icons and so on)
  • FULLSCREEN: full-screen button
