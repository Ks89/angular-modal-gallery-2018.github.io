Default input values

When undefined, modal gallery's inputs have these default values:


Input (.html) Default value (.ts)
modalImages
Mandatory field, without a default value
downloadable
false
buttonsConfig
{ visible: true, strategy: ButtonsStrategy.DEFAULT }
enableCloseOutside
true
previewConfig
{ visible: true, number: 3, arrows: true, clickable: true, size: { height: '50px', width: 'auto' }}
loadingConfig
{enable: true, type: LoadingType.STANDARD}
slideConfig
{ infinite: false, sidePreviews: { show: true, size: { width: '100px', height: 'auto' } } }
accessibilityConfig
an object with all default english accessibility values. Check below for the complete object.
description
{ strategy: DescriptionStrategy.ALWAYS_VISIBLE, imageText: 'Image ', numberSeparator: '/', beforeTextDescription: ' - ' }
keyboardConfig
esc, →, ←
plainGalleryConfig
{ strategy: PlainGalleryStrategy.ROW, layout: this.defaultLayout, advanced: { aTags: false, additionalBackground: '50% 50%/cover' } }
